GitHub 文件加速

GitHub 文件链接带不带协议头都可以，支持 release、archive 以及文件，右键复制出来的链接都是符合标准的。

release、archive 使用 cf 加速，文件会跳转至 JsDelivr

注意，不支持项目文件夹

分支源码：https://github.com/cnrot/project/archive/master.zip

release 源码：https://github.com/cnrot/project/archive/v0.1.0.tar.gz

release 文件：https://github.com/cnrot/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/cnrot/project/blob/master/filename
